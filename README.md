# 🛒 NovaCart - MERN E-Commerce Platform

NovaCart is a full-stack E-Commerce web application built using the MERN stack (MongoDB, Express.js, React.js, and Node.js). The platform provides a seamless online shopping experience with secure authentication, product management, shopping cart functionality, order processing, and an admin dashboard.

## 🚀 Features

### User Features

* User Registration & Login
* JWT Authentication & Authorization
* Browse Products
* Product Search & Filtering
* Product Details Page
* Add to Cart
* Update Cart Quantity
* Place Orders
* View Order History
* Product Reviews & Ratings
* Responsive Design

### Admin Features

* Admin Dashboard
* Manage Products
* Add New Products
* Edit Product Information
* Delete Products
* Manage Users
* Manage Orders
* Update Order Status

## 🛠️ Tech Stack

## ⚙️ Installation

### Clone Repository

bash
git clone https://github.com/your-username/novacart.git
cd novacart

### Backend Setup

bash
cd backend
npm install
npm start

### Frontend Setup

bash
cd frontend
npm install
npm run dev


## 🔐 Environment Variables

Create a `.env` file inside the backend folder and configure:

env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
